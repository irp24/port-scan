# portscan 0.1.3

Simple port scanner

Usage: port-scan [OPTIONS]



Options:

  -i, --ip <IP>  Target ip address [default: 127.0.0.1]
  
  -t, --to <TO>  Connection timeout in ms [default: 250]
  
  -c, --ct <CT>  thread count [default: 100]
  
  -h, --help     Print help
  
  -V, --version  Print version
